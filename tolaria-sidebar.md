# The Sidebar

The sidebar is your primary navigation tool. It groups notes in different ways so you can find what you need quickly.

## Inbox
Notes that aren't organized yet. This is inspired by Tiago Forte's concept of separating capture from organize. Review your inbox weekly to keep it from growing too large. You can disable the inbox in Settings if you prefer to organize notes immediately.

## All Notes
The entire vault, every single note. You can sort by any property—created date, modified date, status, or any custom property you've added.

## Archive
A permanent home for notes you don't want to delete but don't need to see regularly. Archive aggressively to keep your vault clean. Archived notes still appear in search results.

## Favorites
Notes you've manually pinned to keep them top-of-mind. Great for active projects, journal entries, or reference notes you access frequently.

## Types
The main organizational device in Tolaria. Each type has its own icon and color. Click any type to see all notes of that type in the note list. Types include Project, Responsibility, Procedure, Topic, Event, Person, and Note.

## Folders
A fallback for vaults that use folder structure. Tolaria stores notes flat by default, but if you have nested folders, they'll appear here. Listed last because folders are a secondary organization method—not the primary way to organize.

## Note on Trash
Trash was removed from Tolaria. Deleting a note is permanent—but your git history is your safety net. You can always recover deleted notes from git if needed.
